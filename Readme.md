# TexWizard Packer
TexWizard Packer is a fork of [Binarius v2.10.4](https://github.com/nlgxzef/Binarius/releases/tag/v2.10.4) that makes the creation of texture packs for [TexWizard](https://github.com/R-033/texwizard) easier.

## Features
- Automatic management of TexWizard.json and meta.json files
- Automatic batch texture attribute copy from original textures
- Automatic texture renaming on import
- Easier to use & optimized Texture Editor interface
- Extended MainKeys files with more hashes, by [Felipe379](https://github.com/Felipe379/NFSRaider)
- Correct texture names on export
- Easier way to open individual files, just like NFS-TexEd

## How to Use
Check out [TexWizard Wiki Page](https://github.com/R-033/texwizard/wiki) for usage.
