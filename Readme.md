# TexWizard Packer
TexWizard Packer is a tool for editing TexWizard texture packs for Black Box Need for Speed games, powered by [Darius](https://github.com/nlgxzef/Darius "Darius") and [Endscript](https://github.com/nlgxzef/Endscript "Endscript").


## Features
- Automatic management of texture pack meta.json file
- Automatic texture attribute management
